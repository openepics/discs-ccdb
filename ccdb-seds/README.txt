
Summary
=======
The seds-japi is the API for the SEDS protocol in Java.

Components
===========
org.openepics.seds.api:     Package containing the API (interfaces)
org.openepics.seds.core:    Package containing the API (implementation)
org.openepics.seds.util:    Package containing utilities

Pre-requisites
==============
The schemas in "protocol\schema\" are added as resources in the
directory "seds-japi\src\main\resources\schema\".
The version field in org.openepics.seds.util.ValueUtil is accurate and
matches the version of the schema.
